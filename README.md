This project is built with React and Tailwind CSS. It is designed to ensure compatibility across various browsers and devices. The application also includes structured folders, optimized assets, and custom graphics to provide a high-quality user experience.

Key Features

1. Technology Stack

React: For building the user interface with reusable components.

Tailwind CSS: For utility-first, responsive styling.

Autoprefixer: Ensures browser compatibility by adding necessary vendor prefixes automatically.

2. Images and Assets

All images are sourced from Pexels, ensuring they are free of copyright issues.

All company logos and icons are custom-designed in Figma to maintain quality and avoid copyright conflicts.

All images used in the project are in PNG format for optimal quality and transparency.

4. Custom Graphics
